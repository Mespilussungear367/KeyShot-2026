## 💡 Rendering Tips
*   **Use High Resolution:** Set your output resolution to match your final goal.
*   **Adjust Samples:** More samples mean higher quality but longer render times. Use lower samples for previews and higher samples for final images.
*   **Save Often:** Save your work as a scene file frequently to avoid data loss.
*   **Organization:** Group parts of your model in the scene tree tab. This helps when you apply materials to many pieces at once.

## 🔍 Troubleshooting
Sometimes issues arise during setup. Read these solutions if you face trouble.

*   **System Slows Down:** Close other programs that use lots of memory, like web browsers or video editors, while rendering.
*   **Graphics Errors:** Update your graphics card driver from the website of your card manufacturer.
*   **Missing Files:** If the library feels empty, go to the Edit menu, then Preferences, then Folders to ensure the software points to the correct local folders.
*   **File Import Failure:** Ensure your 3D model file format works with KeyShot. Export your model as an OBJ, STEP, or IGES file from your CAD software if it does not open directly.
